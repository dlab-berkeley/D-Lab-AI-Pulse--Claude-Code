# DSGE.jl Source

The DSGE.jl codebase used in this demo is the Federal Reserve Bank of New York's
Dynamic Stochastic General Equilibrium (DSGE) model implementation in Julia.

## Download

**GitHub Repository:**
https://github.com/FRBNY-DSGE/DSGE.jl

Download and extract the repository to this folder as `DSGE.jl-main/`.

## About

DSGE.jl is a Julia package for solving, estimating, and forecasting DSGE models.
It was developed by the Research Group at the Federal Reserve Bank of New York.

## Size Note

The full repository is too large to include in this workshop distribution.
